# AI Prompt Organizer

![スクリーンショット](images/ss.jpg)

## 機能概要

**AI Prompt Organizer** は、画像生成AI・動画生成AIなどで使うプロンプトを、タグ・説明・画像付きで整理できるWindows向けの管理ツールです。

Googleドキュメントやテキストファイルにプロンプトを溜めていくと、あとから探すのが大変になりがちです。
このツールでは、プロンプトを1件ずつカードのように登録して、検索・タグ絞り込み・画像サムネイル付きで管理できます。

### 主な機能

* Windows用GUIアプリ
* タイトル、プロンプト、ネガティブ／補助プロンプト、説明メモをまとめて管理
* 画像を複数登録可能
* 1枚目の画像をサムネイルとしてリスト表示
* 画像ファイルのドラッグ＆ドロップ追加に対応
* タグチップUIで、プロンプトを分かりやすく分類
* タグ絞り込みで目的のプロンプトをすぐ検索
* タグ管理画面で、タグ名・カテゴリ・色を編集可能
* タグプリセット管理に対応
* 使用AI、モデル名、プロジェクト名、評価、お気に入りを記録可能
* プロンプト本文をワンクリックでコピー
* タイトル＋プロンプトのまとめコピーにも対応
* 文字サイズの変更に対応
* SQLiteでローカル保存
* 登録画像はアプリフォルダ内の `assets` に保存
* FFmpeg不要

### 一言で言うと

「AI生成用プロンプトを、画像付きで探しやすく整理するツール」

## 使い方

1. **アプリを起動する**

   配布ZIPを展開して、以下のEXEを起動します。

   ```text
   ai-prompt-organizer/ai-prompt-organizer.exe
   ```

   Pythonやpip installは不要です。

2. **プロンプトを登録する**

   左上の **新規** ボタンを押して、新しいプロンプトを作成します。

   登録できる主な内容は以下です。

   * タイトル
   * 使用AI
   * モデル名
   * プロジェクト名
   * タグ
   * メインプロンプト
   * ネガティブ／補助プロンプト
   * 説明／メモ
   * 評価
   * お気に入り

3. **画像を追加する**

   詳細画面の画像エリアに画像ファイルをドラッグ＆ドロップすると、そのプロンプトに画像を追加できます。

   登録した画像はアプリ側の `assets` フォルダにコピーされます。
   元画像を直接書き換えるわけではありません。

4. **タグを設定する**

   タグ欄にタグ名を入力して追加できます。
   よく使う組み合わせは、タグプリセットとして登録しておくと便利です。

   例：

   * 画像
   * 動画
   * ポーズ
   * キャラ
   * 背景
   * ライティング
   * ネガティブ
   * Layer Breaker
   * ちゃっぴー
   * 成功
   * 要修正

5. **検索・絞り込みをする**

   左上の検索ボックスで、タイトルやプロンプト本文を検索できます。

   左側のタグ絞り込みを使うと、指定したタグを持つプロンプトだけを表示できます。
   お気に入りだけを表示することもできます。

6. **プロンプトをコピーする**

   詳細画面上部のボタンから、登録したプロンプトをコピーできます。

   * **プロンプトをコピー**
   * **タイトル＋プロンプトをコピー**

   生成AIに貼り付けるときに、すぐ使えるようにしています。

7. **保存する**

   編集した内容は **保存** ボタンで保存します。

   データはアプリと同じフォルダ内に作成されるSQLiteデータベースに保存されます。

   ```text
   prompt_organizer.db
   assets/
   ```

## おすすめ設定例

### 画像生成プロンプト用

タグ例：

```text
画像, キャラ, ポーズ, 表情, 背景, 成功
```

### 動画生成プロンプト用

タグ例：

```text
動画, 動画モーション, カメラ, 構図, 要修正
```

### キャラクター管理用

タグ例：

```text
キャラ, ちゃっぴー, 衣装, 表情, ポーズ
```

### 失敗例の管理

失敗したプロンプトも削除せず、以下のようなタグを付けて残しておくと便利です。

```text
要修正, 指が壊れる, 構図は良い, 色味は良い, 没
```

あとから「何が悪かったか」を見返せるので、プロンプト改善に使えます。

## 必要環境

* Windows 10 / 11
* Python不要
* FFmpeg不要
* 追加インストール不要

配布されているEXE版を使う場合、PythonやPySide6をインストールする必要はありません。

## データ保存について

このツールはローカル環境で動作します。
登録したプロンプトや画像情報は、アプリと同じフォルダ内に保存されます。

主な保存先：

```text
prompt_organizer.db
assets/
```

大事なプロンプトを管理する場合は、アプリフォルダごとバックアップしておくのがおすすめです。
