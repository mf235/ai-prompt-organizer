AI Prompt Organizer
===================

AI Prompt Organizer は、AI生成用のプロンプトをカード形式で整理するためのローカルGUIツールです。
タイトル、タグ、説明、素材画像・動画・ファイルなどをまとめて管理できます。


■ インストール方法
------------------

1. 配布ZIPを任意のフォルダに展開してください。

   例:
   C:\Apps\ai-prompt-organizer\

2. 展開したフォルダ内の exe ファイルを起動してください。

   Python のインストールは不要です。

3. 初回起動時に、設定ファイルやデータベースが自動作成されます。

   アプリ本体、assets フォルダ、データベース、設定ファイルは同じフォルダ内で管理されます。
   フォルダごと移動・バックアップすると扱いやすいです。


■ アップデート方法
------------------

1. 念のため、現在のフォルダをバックアップしてください。

2. 新しいZIPを展開します。

3. 既存のデータを引き継ぐ場合は、必要に応じて以下を新しいフォルダへコピーしてください。

   - assets フォルダ
   - データベースファイル
   - 設定ファイル
   - _backup フォルダ

※ ファイル構成はバージョンによって変わる場合があります。
※ 大事なデータがある場合は、必ずバックアップしてから更新してください。


■ アンインストール方法
----------------------

レジストリへの特別な登録は行いません。
不要になった場合は、展開したフォルダごと削除してください。

スタートアップ登録を使用している場合は、アプリ内設定から解除してから削除してください。


■ 動作環境
----------

- Windows 10 / 11
- exe版は基本的にそのまま起動できます
- 起動しない場合は、Microsoft Visual C++ Redistributable の導入が必要になる場合があります


■ 注意事項
----------

- このツールはローカル環境で動作します。
- 管理しているプロンプトや素材ファイルは、ユーザー自身のPC上に保存されます。
- 重要なデータは定期的にバックアップしてください。
- 大量の画像・動画素材を登録すると、フォルダサイズが大きくなる場合があります。
